# 可见光通道短视频后台播放问题修改记录

1. 问题原因：HuanYuDevice通道一录制时，MP4头部SPS/PPS信息不完整，同时视频sample中混入H264配置帧，后台播放器只显示首帧后无法继续播放。
2. MyOnvifDevice修改：初始化MediaMuxer时优先使用实时RTP解析出的SPS/PPS，并补齐H264 start code，保证avcC信息完整。
3. Device修改：写入MP4视频sample时跳过NAL type 6/7/8，只写入真实视频帧，关键帧继续按NAL type 5标记。
4. 验证结果：本地播放器和后台均可正常播放新录制的可见光通道短视频。
